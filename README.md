## 📌 Project Overview

This project predicts **Credit Card Approval** using Machine Learning. A **Random Forest Classifier** is trained on applicant financial and personal data to determine whether a credit card application should be approved or rejected.

### Key Features
- ✅ Real-time prediction via a Flask web application
- ✅ Random Forest Classifier (200 estimators) with ~85%+ accuracy
- ✅ Clean, responsive web UI
- ✅ Auto-trains model if not already saved
